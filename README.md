# gonogo
**Go/No-Go Task** (PsychoPy)

This project implements a simple Go/No-Go task using PsychoPy. The task is based on the paradigm described in Price et al. (2016) to investigate inhibition response with food stimuli.

Participants are instructed to press the spacebar right after GO cue and skip responding on NO-GO cue. Each trial comes after a fixation cross, a cue indicating GO or NO GO, and then a food image stimulus.

**Task Structure**
Fixation cross (500 ms)
GO / NO GO cue (500 ms)
Food image stimulus (750 ms)
Blank screen (500 ms)
Healthy food images are used for GO trials, and high-fat food images are used for NO-GO trials.

Files Included

go_nogo.py – main PsychoPy experiment script

HF_LF_60.csv – list of food stimuli and attributes

Food-Choice-Task-main (1).zip - contains 'stimuli'folder, which has food images used in the experiment

**How to Run**
Open go_nogo.py. on psychopy
Run the script.
Complete the Go/No-Go task.
Output
The script saves a CSV file as a result for each run.
